Frequently Asked Questions

Can I use this app for free?

Yes, you can once you sign up without limitations.

Can I rely on the data?

The data is taken from three sources: Sunrise Sunset web ("https://sunrise-sunset.org/api"), google maps and wikipedia.

Can I use part or the entire code in my projects?

Yes, you can. No copyrigths here. Anycase it would be greateful if you mention the author.

Do you plan to restrict access in the future?

I don't plan to restrict access today and I doubt in the future.

Are there other applications that give the same information?

I don't know. It's quite possible that there are because the web is very huge.
